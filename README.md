# Wavelength Web Game

A web-based adaptation of the board game **Wavelength**, built with HTML, CSS, and JavaScript. The project recreates the social guessing experience where players interpret clues within a spectrum between two opposite concepts.

## Features
- Interactive dial and spectrum system
- Competitive and cooperative modes
- Team and score management
- Responsive UI and animated effects
- Custom themes and cards
- Local multiplayer experience

## Technologies
- HTML5
- CSS3
- JavaScript
